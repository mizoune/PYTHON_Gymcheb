# Lekce 1: Proměnné v Pythonu

Ahoj! Dnes se naučíme, jak Python ukládá data.

### Teorie
Proměnná je jako krabička, do které si něco schováš:
`jmeno = "Robot"`

### Tvůj úkol
1. Otevři soubor `main.py` v Codespaces.
2. Vytvoř proměnnou `vek` a ulož do ní své stáří.
3. Pomocí `print()` vypiš větu: "Je mi X let".
4. Nezapomeň změny uložit (Commit & Push)!
